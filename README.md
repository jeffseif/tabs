# tabs

CLI tool for ukulele tabs

## Example

```shell
> ./cli --chord=C
chord=C

3         | - | - | ●           (C)
0       ○ | - | - | -           (E)
0       ○ | - | - | -           (C)
0       ○ | - | - | -           (G)
```

```shell
> ./cli --chord=AB7
chord=A♯|B♭7

1         | ● | -               (A♯|B♭)
1         | ● | -               (F)
2         | - | ●               (D)
1         | ● | -               (G♯|A♭)
```

```shell
> ./cli --notes='D G A'
chord=Dsus4

0       ○ | - | - | -           (A)
3         | - | - | ●           (G)
2         | - | ● | -           (D)
0       ○ | - | - | -           (G)
```
