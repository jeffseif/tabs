# tabs

CLI tool for ukulele tabs

## Example

### Show the tab for chords

```shell
> ./cli chords Amin D G Emin7

  Amin

○ | - | - | - | - | - | -       (A)
○ | - | - | - | - | - | -       (E)
○ | - | - | - | - | - | -       (C)
  | - | ● | - | - | - | -       (A)

  D

○ | - | - | - | - | - | -       (A)
  | - | ● | - | - | - | -       (F♯|G♭)
  | - | ● | - | - | - | -       (D)
  | - | ● | - | - | - | -       (A)

  G

  | - | ● | - | - | - | -       (B)
  | - | - | ● | - | - | -       (G)
  | - | ● | - | - | - | -       (D)
○ | - | - | - | - | - | -       (G)

  Emin7

  | - | ● | - | - | - | -       (B)
○ | - | - | - | - | - | -       (E)
  | - | ● | - | - | - | -       (D)
○ | - | - | - | - | - | -       (G)
```

### Show the chords for frets

```shell
> ./cli frets --maximum-count=2 0013

  Csus4

  | - | - | ● | - | - | -       (C)
  | ● | - | - | - | - | -       (F)
○ | - | - | - | - | - | -       (C)
○ | - | - | - | - | - | -       (G)

  Fsus2

  | - | - | ● | - | - | -       (C)
  | ● | - | - | - | - | -       (F)
○ | - | - | - | - | - | -       (C)
○ | - | - | - | - | - | -       (G)
```

```shell
> ./cli notes --maximum-count=4 'C DE FG A'
```
